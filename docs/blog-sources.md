# Build 2026 巡回先ブログ一覧

researcher エージェントがアナウンスの網羅性を確保するための巡回先リスト。
Build 週（6/1-6/6）に投稿された記事のうち「Build 2026」を冠するものが対象。

## 巡回方法

各ブログの RSS or 最新記事一覧を Build 週でフィルタリングし、未カバーの発表を特定する。

## メンテナンスポリシー

- 新しいブログボードが発見されたら追加する
- 巡回で Build 関連記事が見つからなかったボードは優先度を下げてよい（削除はしない）
- research スキルの探索カテゴリとこのリストは同期させる
- DevBlogs の全一覧は `https://devblogs.microsoft.com/404-page` の "All Blogs" セクションで確認可能
- TechCommunity のハブ一覧は `https://techcommunity.microsoft.com/Directory` で確認可能（クライアントレンダリング）
